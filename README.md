# Churn-Prediction-with-Cost-Sensitive-Learning
A bank loses a client every day. The real question is not how many — it's which ones.  Losing a client with €1,000 in savings is not the same as losing one with €50,000. Standard churn models treat both errors equally. This one doesn't.  I use Cost-Sensitive Learning — weighting each client by their LTV (Lifetime Value).
